Alcance:


Asegurar una experiencia de usuario satisfactoria a través de la utilización de tecnologías de transmisión de video y audio de alta calidad y la implementación de características como la personalización de perfiles de usuario y la recomendación de contenido. Garantizar la seguridad y privacidad de los datos de los usuarios y cumplir con todas las regulaciones y leyes pertinentes en relación con la protección de datos en línea y como principal alcance poder establecer acuerdos con empresas de medios de comunicación y proveedores de contenido para adquirir los derechos de transmisión para asegurarse de que todos los usuarios de este aplicativo cumplan con todas las regulaciones y leyes pertinentes.

Desarrollo

FRONTEND

Se ha implementado la Biblioteca Thymeleaf como motor de plantilla, ya que se acopla muy bien para trabajar en la capa vista del MVC de aplicaciones web, además puede procesar cualquier archivo XML, incluso en entornos desconectados.

Para realizar el diseño de la interfaz grafica se utilizo bootstrap 5 ya que es una biblioteca multiplataforma o conjunto de herramientas de código abierto para diseño de sitios y aplicaciones web.

BASE DE DATOS EER.

Base de datos del proyecto

Una base de datos entidad-relación (ER) es un modelo de datos utilizado en la gestión de bases de datos para representar la estructura de los datos y las relaciones entre diferentes entidades.

Este modelo se basa en un enfoque visual que utiliza diagramas para representar cómo los datos están organizados y cómo se relacionan entre sí.
